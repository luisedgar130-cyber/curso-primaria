# Curso Primaria

## Bienvenida
Bienvenidos al sitio web del curso de primaria. Este curso está diseñado para proporcionar una educación integral a los estudiantes de primaria, cubriendo materias fundamentales y fomentando un ambiente de aprendizaje dinámico.

## Estructura del Curso
El curso está organizado en módulos que abarcan diversas disciplinas:
- Matemáticas
- Ciencias
- Lengua
- Historia
- Educación Física

Cada módulo contiene recursos, actividades y evaluaciones para ayudar a los estudiantes a comprender y aplicar los conceptos aprendidos.

## Recursos
- **Materiales de lectura**: Disponibles en formato digital y físico.
- **Ejercicios interactivos**: Herramientas en línea para reforzar el aprendizaje.
- **Soporte técnico**: Asistencia disponible para resolver dudas sobre el uso de la plataforma.
