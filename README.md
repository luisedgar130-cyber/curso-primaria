# 🎓 Curso de Primaria

Plataforma educativa interactiva para estudiantes de primaria con 5 módulos completos, sistema de gamificación, actividades y seguimiento de progreso.

## ✨ Características Principales

- 🎮 **Sistema de gamificación** — puntos, niveles, logros y racha de aprendizaje
- 📚 **5 módulos educativos** con teoría, videos y actividades
- ✅ **Actividades interactivas** con retroalimentación inmediata
- 📊 **Dashboard de progreso** con barras visuales
- 💾 **Guardado automático** en localStorage (sin backend)
- 🎊 **Confeti y animaciones** al completar actividades
- 📱 **Totalmente responsivo** — funciona en móvil, tablet y desktop

## 📁 Estructura de Archivos

```
curso-primaria/
├── index.html    — Estructura HTML5 completa
├── styles.css    — Estilos profesionales con variables CSS y animaciones
├── script.js     — Lógica de gamificación, quizzes y localStorage
└── README.md     — Esta documentación
```

## 📚 Módulos del Curso

| Módulo | Contenido | Actividades |
|--------|-----------|-------------|
| 📐 Matemáticas | Números, operaciones, multiplicación | 3 preguntas |
| 🔬 Ciencias Naturales | Plantas, ciclo del agua, sistema solar | 3 preguntas |
| 📖 Español | Gramática, vocales, tipos de texto | 3 preguntas |
| 🏛️ Historia | Civilizaciones, independencia, cultura | 3 preguntas |
| 🏃 Educación Física | Deportes, salud, alimentación | 3 preguntas |

## 🎯 Sistema de Puntuación

| Acción | Puntos |
|--------|--------|
| Respuesta correcta | +10 ⭐ |
| Respuesta incorrecta | −5 |
| Módulo completado | Toast de celebración 🎉 |
| Logro desbloqueado | Confeti + notificación |

## 🚀 Cómo Usar

### Opción 1 — GitHub Pages (Recomendado)
1. Activa GitHub Pages en **Settings → Pages → Branch: main**
2. Accede a: `https://luisedgar130-cyber.github.io/curso-primaria/`

### Opción 2 — Local
1. Descarga o clona el repositorio
2. Abre `index.html` directamente en tu navegador

### Opción 3 — Servidor Local
```bash
# Con Python 3
python -m http.server 8000
# Abre http://localhost:8000
```

## 🎨 Diseño

- **Colores**: Azul (#2C3E50), Rojo (#E74C3C), Verde (#27AE60)
- **Tipografía**: Segoe UI / Arial (fuentes del sistema)
- **Animaciones**: Transiciones de 0.3s, confeti, hover effects
- **Layout**: CSS Grid + Flexbox, totalmente responsivo

## 💻 Tecnologías

- **HTML5** semántico
- **CSS3** con variables, grid, animaciones y media queries
- **JavaScript (ES6+)** vanilla, sin dependencias externas
- **localStorage** para persistencia de datos
